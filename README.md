# Choose Your Own Adventure Game 🌟

A text-based adventure game built using Python where players make choices that determine the outcome of the story.

## Features
- Interactive storyline
- Multiple paths and endings
- Choice-based gameplay
- User input handling
- Replay option to play again

## Technologies Used
- Python
- Conditional Statements (if-elif-else)
- Loops
- User Input Handling

## Learning Outcomes
- Creating interactive programs
- Using decision-making logic
- Managing multiple outcomes
- Working with loops and conditions

## How to Run
1. Open the project in Python.
2. Run the program.
3. Follow the story and make choices.
4. Reach different endings based on your decisions.

## Story Overview
The player starts on a dirt road and must choose different paths. Along the journey, they face rivers, bridges, and strangers. Every decision affects the final outcome of the adventure.

## Sample Gameplay

Welcome to the Adventure Game!

You are on a dirt road.
You can go left or right.

> right

You come to a bridge.
Do you want to cross it or go back?

> cross

You meet a stranger.
Do you want to talk to them?

> yes

Congratulations! You found treasure and won the game!
